## 필수 섹션

모든 상세페이지에 포함되어야 하는 섹션:

1. **한눈에 보기** - 카테고리, 난이도, 기간(4주), 도구, 결과물
2. **이런 분께 추천해요** - 대상 (구체적 상황 + 니즈, 3-4개)
3. **이런 문제를 바이브코딩으로 해결해요** - 문제 → 해결 (3-4개)
4. **바이브코딩으로 만드는 결과물** - 4주 후 산출물 (구체적으로)
5. **커리큘럼** - 4주 학습 흐름 (주차별 상세)
6. **바이브코딩 도구 소개** - 사용할 도구 설명
7. **스터디장 소개** - 이력 (공동스터디장 있으면 함께)

## 조건부 섹션

### 중급인 경우

"한눈에 보기" 다음에 **참여 조건** 섹션 추가:

```markdown
## 참여 조건 (필수)

📹 **사전 학습 영상**
- [영상 제목](URL) - 예상 시청 시간
- [영상 제목](URL) - 예상 시청 시간

📚 **선수지식**
- [구체적인 선수지식 항목]
- [구체적인 선수지식 항목]

⚠️ 위 조건을 충족해야 스터디 참여가 가능합니다.
```

### 입문이지만 사전지식 추가 시

"한눈에 보기" 다음에 **미리 알면 좋아요** 섹션 추가:

```markdown
## 미리 알면 좋아요 (선택)

📹 **참고 영상**
- [영상 제목](URL)

📚 **기본 개념**
- [알면 도움되는 내용]

💡 필수는 아니지만, 미리 보고 오시면 스터디가 더 수월해요!
```

## 제목 생성 규칙

형식: `[대상]을 위한 [도구]로 [결과물] 만들기`

**예시:**
- 비개발자를 위한 Claude Code로 업무 자동화 스킬 만들기
- 1인 사업자를 위한 Cursor로 랜딩페이지 만들기
- 마케터를 위한 Lindy로 캠페인 자동화 에이전트 만들기
- 커뮤니티 운영자를 위한 OpenCode로 자동화 시스템 만들기

| 항목 | 설명 |

| 주제 | 그 주차의 핵심 키워드 |
| 학습 목표 | 이번 주에 달성할 것 |
| 주요 내용 | 구체적인 학습 항목 (3-4개) |

## 예시 상세페이지

```markdown
# 비개발자를 위한 Claude Code로 업무 자동화 스킬 만들기

## 한눈에 보기

| 항목 | 내용 |
|------|------|
| 카테고리 | 개발&자동화 |
| 난이도 | 입문 🐥 |
| 기간 | 4주 |
| 도구 | Claude Code |
| 결과물 | 나만의 업무 자동화 스킬 1개 |

## 이런 분께 추천해요

- 반복 업무에 지친 비개발자
- AI로 뭔가 만들어보고 싶은데 코딩은 못하는 분
- Claude Code가 뭔지 궁금한 분
- 터미널이 무섭지만 도전해보고 싶은 분

## 이런 문제를 바이브코딩으로 해결해요

| 기존 문제 | 바이브코딩 해결 |
|----------|----------------|
| 매일 같은 보고서 작성 | 스킬 하나로 자동 생성 |
| 파일 정리 노가다 | 명령 한 줄로 자동 분류 |
| 데이터 취합 반복 작업 | 스킬로 자동 수집 + 정리 |

## 바이브코딩으로 만드는 결과물

4주 후 가져갈 것:
- **나만의 업무 자동화 스킬 1개** - 실제 내 업무에 적용 가능
- **Claude Code 기본 사용법** - 터미널 공포증 극복
- **MCP 연동 경험** - 외부 서비스와 연결하는 법

## 커리큘럼

### 1주차: 환경 세팅 & 첫 만남
**학습 목표**: Claude Code 설치하고 첫 대화 나누기

| 항목 | 내용 |
|------|------|
| 주요 내용 | • Claude Code 설치 및 인증<br>• 터미널 기본 명령어 (cd, ls, pwd)<br>• 첫 번째 대화: "안녕 Claude!" |
| 실습 과제 | Claude Code로 간단한 파일 만들어보기 |
| 결과물 | 작동하는 Claude Code 환경 |

### 2주차: 스킬의 세계로
**학습 목표**: 스킬이 뭔지 이해하고 간단한 스킬 만들기

| 항목 | 내용 |
|------|------|
| 주요 내용 | • 스킬이란? (SKILL.md 구조)<br>• 기존 스킬 분석해보기<br>• 나만의 첫 스킬 작성 |
| 실습 과제 | "오늘 날씨 알려줘" 스킬 만들기 |
| 결과물 | 작동하는 간단한 스킬 1개 |

### 3주차: 내 업무에 적용하기
**학습 목표**: 실제 내 반복 업무를 자동화하는 스킬 개발

| 항목 | 내용 |
|------|------|
| 주요 내용 | • 내 반복 업무 분석<br>• 자동화 대상 선정<br>• 스킬 설계 및 개발 |
| 실습 과제 | 내 업무 자동화 스킬 초안 작성 |
| 결과물 | 업무 자동화 스킬 초안 |

### 4주차: 완성 & 공유
**학습 목표**: 스킬 완성하고 다른 사람과 공유하기

| 항목 | 내용 |
|------|------|
| 주요 내용 | • 스킬 테스트 및 디버깅<br>• 스킬 문서화<br>• 공유 및 피드백 |
| 실습 과제 | 완성된 스킬로 실제 업무 처리해보기 |
| 결과물 | 완성된 업무 자동화 스킬 1개 |

## 바이브코딩 도구 소개

**Claude Code**는 터미널에서 자연어로 코딩하는 CLI 도구입니다.
코드를 직접 쓰지 않아도 "이런 스킬 만들어줘"라고 말하면 Claude가 만들어줍니다.
```
