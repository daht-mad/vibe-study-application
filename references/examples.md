## 카테고리별 바이브코딩 전환 예시

기존 스터디 주제를 바이브코딩 방향으로 전환하는 안내.

### 콘텐츠&지식

| 세분류 | 기존 주제 | 바이브코딩 전환 |
|--------|-----------|-----------------|
| 영상·숏폼 | 유튜브 자막/썸네일 제작 | OpenCode로 자막 자동 생성 + 메타데이터 관리 |
| 디자인·브랜딩 | Canva로 브랜드 에셋 제작 | Claude Code로 디자인 가이드라인 자동 적용 |
| 글쓰기·출판 | GPT로 전자책 집필 | Antigravity에서 md 노트 모아서 집필 + 원클릭 배포 |
| 교육·지식관리 | ChatGPT로 수업자료 제작 | Cursor로 학습 자료 자동 생성 + 노션 연동 |

### 업무&비즈니스

| 세분류 | 기존 주제 | 바이브코딩 전환 |
|--------|-----------|-----------------|
| 창업·BM설계 | ChatGPT로 시장 조사 | OpenClaw로 시장 조사 자동화 + 경쟁사 분석 |
| 마케팅·퍼널 | Make로 마케팅 캠페인 자동화 | Lindy로 캠페인 자동 생성 에이전트 |
| 커머스·랜딩 | Gamma로 랜딩페이지 제작 | Google Build로 상품 설명 + 랜딩페이지 생성 |
| 데이터·CRM | 노션/에어테이블 기반 CRM | OpenCode 스킬로 데이터 분석 + 에어테이블 연동 |

### 개발&자동화

| 세분류 | 기존 주제 | 바이브코딩 전환 |
|--------|-----------|-----------------|
| 업무자동화 | Activepieces/n8n으로 리포트 자동화 | Claude Code 스킬로 워크플로우 자동화 + MCP 활용 |
| 웹앱·개발 | 노코드로 MVP 만들기 | Cursor로 코드 생성 + 테스트 자동화 |
| 챗봇·에이전트 | 카톡봇 + 자동화 서버 | Manus로 에이전트 시스템 구축 |
| 새로운 툴 | AI 도구 탐색 스터디 | OpenClaw로 새로운 MCP 탐색 + 스킬 확장 |

## 도구별 사전학습 추천

**중급 선택 시** AI가 도구에 맞는 사전학습을 추천할 때 참고.

| 도구 | 추천 사전학습 영상 | 추천 선수지식 |
|------|-------------------|--------------|
| Cursor | Cursor 공식 튜토리얼, 바이브코딩 입문 영상 | IDE 기본 사용법, 프롬프트 작성법 |
| Claude Code | Claude Code 설치 가이드, MCP 기초 | 터미널 기본 명령어, CLI 환경 이해 |
| OpenCode | OpenCode 시작하기 | 터미널 기본, AI 코딩 도구 경험 |
| Lindy | Lindy 공식 튜토리얼 | 워크플로우 자동화 개념 |
| OpenClaw | OpenClaw 입문 | PC 자동화 기초, 에이전트 개념 |
| Manus | Manus 시작 가이드 | 에이전트 기반 자동화 이해 |
| Antigravity | Antigravity 공식 문서 | 마크다운 기초, 웹 배포 개념 |

```
- [도구 기본 설치 및 설정 방법]

이대로 괜찮으시면 "확인"이라고 해주시고,

```
