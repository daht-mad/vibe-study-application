# 22기 바이브코딩 스터디 — 도구·레벨·슬랏 레퍼런스

인터뷰 시 이 문서를 참고하여 지원자에게 도구 추천, 레벨 안내, 주제 예시를 제공한다.

---

## 도구 소개

지원자가 도구를 모를 수 있으니, 카테고리+레벨에 맞는 도구를 추천할 때 이 설명을 참고.

### 입문용 도구 (노코드/대화형)

| 도구 | 설명 |
|------|------|
| **Google AI 스튜디오 빌드** | Google의 AI 앱 빌더. 코딩 없이 자연어로 앱을 만들 수 있다. 입문자가 첫 앱을 만드는 데 적합. |
| **Claude 코워크** | Anthropic의 AI 비서. 데스크톱에서 로컬 파일/앱에 접근해서 업무를 도와준다. 이메일 정리, 회의록 요약, 보고서 작성 등 대화형으로 업무 자동화. |
| **NotebookLM** | Google의 AI 학습 도구. 문서를 넣으면 AI가 대화형 팟캐스트, 요약, Q&A를 생성. 콘텐츠 제작·지식관리 입문에 적합. |

### 중급용 도구 (코드 터치)

| 도구 | 설명 |
|------|------|
| **Claude Code** | Anthropic의 AI 코딩 에이전트. 터미널에서 자연어로 지시하면 코드를 작성·수정·배포. 멀티파일 작업, 리팩토링, 테스트 자동 생성 가능. 개발자 선호도 1위. |
| **Cursor** | AI가 내장된 코드 에디터(IDE). 코드 작성 중 AI가 실시간으로 도와준다. 탭 자동완성, Agent Mode로 복잡한 기능 구현 가능. |

### 고급용 도구 (시스템 설계)

| 도구 | 설명 |
|------|------|
| **OpenClaw (오픈클로)** | 지피터스가 만든 멀티에이전트 AI 플랫폼. YAML+마크다운 설정으로 AI 에이전트를 만들어 Slack, Telegram, KakaoTalk 등에서 24/7 자동 운영. 코드를 짜는 게 아니라 에이전트의 성격·도구·워크플로우를 설계한다. |
| **MCP (Model Context Protocol)** | Anthropic이 만든 AI 도구 연결 표준. "AI용 USB-C"라고 불린다. MCP 서버를 만들면 Claude Code 등 AI 도구가 내 Notion, Airtable, 캘린더 등에 직접 접근 가능. |
| **하네스 엔지니어링** | AI 에이전트를 감싸는 실행 환경을 설계하는 방법론. CLAUDE.md 같은 프로젝트 지시 파일, 도구 접근 권한, 검증 루프, 가드레일을 설계하여 에이전트 성능을 극대화. |

## 레벨별 안내

### 입문 🐥 — AI랑 대화만으로 만든다

- **도구**: 노코드/대화형 (AI 스튜디오 빌드, 코워크, NotebookLM)
- **대상**: AI 도구 처음 써보는 사람
- **결과물**: 나 혼자 쓰는 결과물 1개 완성
- **코드 비율**: 0% (대화만으로)
- **사전학습**: 선택 (있으면 좋지만 필수 아님)

### 중급 — 코드랑 친해지면서 진짜 서비스를 만든다

- **도구**: 코드 터치 (Cursor, Claude Code)
- **대상**: AI 도구 써봤고 실무 적용하려는 사람
- **결과물**: 다른 사람도 쓸 수 있는 서비스/시스템
- **코드 비율**: 20~50% (AI가 짜고 내가 수정)
- **사전학습**: 필수 (영상 URL + 선수지식)

### 고급 — AI가 알아서 돌아가는 시스템을 설계한다

- **도구**: 시스템 설계 (에이전트, MCP, 오케스트레이션)
- **대상**: 이미 AI를 적용 중이고 시스템화/확장하려는 사람
- **결과물**: 사람 없이 자동으로 돌아가는 시스템
- **코드 비율**: 설계 중심 (아키텍처 결정 + AI 실행)
- **사전학습**: 필수 (영상 URL + 선수지식)

## 카테고리 × 레벨 슬랏별 주제 예시

지원자의 카테고리+레벨 조합에 맞는 주제를 추천할 때 참고.

### 개발&자동화

| 레벨 | 슬랏 A | 슬랏 B |
|------|--------|--------|
| **입문** | AI 스튜디오 빌드로 내 앱 만들기 (예약 앱, 포트폴리오, 일정 관리) | 코워크로 반복 업무 자동화 (엑셀 정리, 이메일 분류, 회의록 추출) |
| **중급** | Claude Code로 수익형 웹서비스 배포 (SaaS, 견적서 도구, 출결 시스템) | Cursor로 업무 도구 만들기 (대시보드, 슬랙 봇, 크롬 확장프로그램) |
| **고급** | 오픈클로로 24/7 AI 에이전트 시스템 (CS봇, 커뮤니티 관리, 주문 처리) | 하네스 엔지니어링 (CLAUDE.md 설계, MCP 서버, 멀티에이전트 오케스트레이션) |

### 콘텐츠&지식

| 레벨 | 슬랏 A | 슬랏 B |
|------|--------|--------|
| **입문** | NotebookLM으로 AI 팟캐스트/학습 콘텐츠 (전공 팟캐스트, 뉴스 큐레이션) | AI로 숏폼 영상 만들기 (지식 해설, AI 보이스+애니메이션, 제품 리뷰) |
| **중급** | Claude Code로 지식관리 시스템 (옵시디언+AI 자동 태깅, 팀 지식베이스) | AI 콘텐츠 반자동 발행 (뉴스레터, 블로그 초안, SNS 캘린더) |
| **고급** | RAG로 조직 전용 AI 검색 시스템 (사내 위키, 법률 문서, 기술 문서) | AI 에이전트 운영 콘텐츠 채널 (풀자동 발행, AI 큐레이터, 다국어) |

### 업무&비즈니스

| 레벨 | 슬랏 A | 슬랏 B |
|------|--------|--------|
| **입문** | 코워크로 매일 2시간 아끼기 (이메일 초안, 회의록, 보고서, 엑셀 탈출) | AI로 퍼스널브랜딩 시작 (톤앤매너 글쓰기, 포트폴리오, SNS 콘텐츠) |
| **중급** | Claude Code로 팀 업무 자동화 (슬랙 봇, 채용 스크리닝, 리드 스코어링) | AI 마케팅 자동화 (콘텐츠 자동 발행, DM 응대, 광고 소재 A/B) |
| **고급** | 1인 사업자 AI 풀스택 운영 (리드→상담→계약→정산, 온라인 강의 사업) | 오픈클로로 고객 여정 풀퍼널 자동화 (온보딩→리텐션→업셀 에이전트) |

## 도구별 사전학습 추천

**중급/고급 선택 시** AI가 도구에 맞는 사전학습을 추천할 때 참고.

| 도구 | 추천 사전학습 | 추천 선수지식 |
|------|-------------|-------------|
| Cursor | Cursor 공식 튜토리얼, 바이브코딩 입문 영상 | IDE 기본 사용법, 프롬프트 작성법 |
| Claude Code | Claude Code 설치 가이드, MCP 기초 영상 | 터미널 기본 명령어, CLI 환경 이해 |
| OpenClaw | 지피터스 오픈클로 소개 영상, 뽀야의 서재(bboya-viewer.vercel.app) | 에이전트 개념, Slack 봇 경험, YAML/마크다운 기초 |
| MCP | MCP 공식 문서, Claude Code MCP 서버 만들기 튜토리얼 | Claude Code 사용 경험, API 기초 개념 |
| 하네스 엔지니어링 | Martin Fowler 하네스 엔지니어링 아티클, CLAUDE.md 작성법 | Claude Code 실무 경험, 프롬프트 엔지니어링 |
| RAG | RAG 개념 설명 영상, 벡터DB 기초 | Python 또는 Node.js 기초, API 호출 경험 |

### 사전학습 안내 예시

```
중급/고급을 선택하셨네요! 사전학습이 필수예요.

제가 [도구] 기준으로 추천드릴게요:

📹 사전학습 영상 추천:
- [도구 공식 튜토리얼 또는 관련 유튜브 영상 URL]

📚 선수지식 추천:
- [도구 기본 설치 및 설정 방법]
- [기본적인 사용법 숙지]

이대로 괜찮으시면 "확인"이라고 해주시고,
다른 영상이나 선수지식이 있으시면 알려주세요!
```
